# Notes

## Use this skill for
- choosing where a new file should live
- deciding whether to extend an existing feature slice
- checking whether a proposed refactor fights the current tree
- avoiding duplicate or parallel directory structures

## Default bias
- inspect before editing
- prefer existing slices
- minimize moves
- keep file-system-synced groups stable

## Not for
- open-ended product architecture debates
- rare one-off placement decisions that are obvious from the local file
