# Cash Retrieval

## Load order

1. `AGENTS.md`
2. `REPO_CONTEXT.md`
3. `package.json`
4. `prisma/schema.prisma`
5. `vercel.json`
6. Relevant `src/` path

## High-authority surfaces

- Prisma schema.
- Runtime source.
- Vercel config.
- Wallet/pass integration files.

## Exclude by default

- `.env*`
- certificates and private keys
- generated Prisma output
- build output
